三維空間旋轉矩陣
計算 Pitch (俯仰)、Yaw (偏航)、Roll (翻滾) 的歐拉角旋轉矩陣。
def get_rotation_matrix(p, y, r):
    cx, sx = math.cos(p), math.sin(p)
    cy, sy = math.cos(y), math.sin(y)
    cz, sz = math.cos(r), math.sin(r)
    return np.array([
        [cy*cz + sy*sx*sz, -cy*sz + sy*sx*cz, sy*cx],
        [cx*sz, cx*cz, -sx],
        [-sy*cz + cy*sx*sz, sy*sz + cy*sx*cz, cy*cx]
    ])
三維到二維的透視投影
將 3D 點投影至 2D 畫面的核心是「Z 軸除法」，模擬近大遠小的透視效果。
def project_point(pt, f, cx, cy):
    # x = (X * f / Z) + cx
    # y = (Y * f / Z) + cy
    return (
        int((pt[0] * f / pt[2]) + cx),
        int((pt[1] * f / pt[2]) + cy)
    ) if pt is not None and pt[2] > 10.0 else None
[SAFETY] 設定 `pt[2] > 10.0` (Z軸大於 10mm) 是為了避免物體過於靠近相機時造成 除以零 (Divide by Zero) 的數學崩潰。
線段相交幾何演算法
利用 向量外積 (Cross Product)。若點 A、B 分別在另一條線段的兩側，則外積結果必定一正一負（相乘小於零）。
def intersect(p1, p2, p3, p4):
    # 計算 2D 外積
    c = lambda o, a, b: \
        (a[0] - o[0]) * (b[1] - o[1]) - \
        (a[1] - o[1]) * (b[0] - o[0])

    return (
        (c(p3, p4, p1) * c(p3, p4, p2) < 0) and
        (c(p1, p2, p3) * c(p1, p2, p4) < 0)
    )
抖動抑制：One Euro Filter
直接使用 MediaPipe 輸出的 3D 座標，在微小動作時會產生令人不適的高頻雜訊（抖動）。
一歐元濾波器能「動態調整截止頻率」。靜止時重度濾波，快速移動時輕度濾波降低延遲。
一歐元濾波器數學公式與程式碼
透過計算當前速度 `dx_prev` 即時動態調整 `cutoff` 頻率，決定當下的平滑權重 `a`。
def __call__(self, x, dt):
    # 計算速度 (dx)
    self.dx_prev = a_d * ((x - self.x_prev) / dt) + (1.0 - a_d) * self.dx_prev

    # 依據速度動態調整截止頻率
    cutoff = self.mincutoff + self.beta * math.hypot(*self.dx_prev)

    # 計算平滑權重 a 並更新位置
    a = 1.0 / (1.0 + 1.0 / (2 * math.pi * cutoff * dt))
    self.x_prev = a * x + (1.0 - a) * self.x_prev
    return self.x_prev
CrossGestureDetector 類別設計

class CrossGestureDetector:
    def __init__(self, hold_duration=1.2):
        self.hold_duration = hold_duration
        self._reset()

    def _reset(self):
        self.cross_start_time = 0
        self.last_seen_time = 0
        self.is_crossing = False
        self.progress = 0.0
CrossGestureDetector 的時序判定
當幾何相交成立，系統開始累積 `time.time()` 直到 `progress` 達到 1.0 (100%)。
